# Матеріали до проведення занять розмовного аглійського клубу

Кому це найбільше допоможе: дітина розуміє англійську мову, але не охоче спілкується. Може соромиться, боїться помилитися, або просто "мова не йде".

## Рівні учасників

### 1) Учасник англійську важко розуміє потребує перекладу для розуміння. Почувши переклад, розуміє про що мова

на цьому рівні буде дублювання більшості фраз українською мовою. Учаснику потрібно буде лиш повторювати надані фрази. Важливим моментом буде те що фрази сповнені емоціями. Якщо це клич варвара "Я ВМИТЬ ЗНИЩУ ТОГО ДРАКОНА СВОЄЮ СКРИПКОЮ", то його треба буде озвучити відповідно. Вередлива принцеса мусить сказати "ФЄ, МЕНІ НАБРИДЛІ ЦІ ЛАСТИ, Я ХОЧУ ЇХ ЗНЯТИ" з жестами, з емоцією.

### 2) Учасник розуміє англійську, деякі слова ведучого можуть бути незрозумілими. Але якщо перепитає, то з пояснення розуміє про що мова.

На цьому рівні дублювання українською мовою буде мінімізовано. Більшість пояснень відбувається англійською, можлива допомога учаснику з тим щоб придумати, фразу для персонажа. Учасник намагається сказати що певний набір елементів не має сенсу.

### 3) Учасник відповідає на питання, обгрунтовує свої відповіді, надає пояснення своїм діям.

На цьому рівні відповіді ще перекладаються у голові. Інколи через те можливe плутання думок через спроби пов'язати українську з англійською. Якщо не виходить, то буває пригальмовуєтсья мовлення.
Учасник більше відповідає. Якщо сам має щось подумати то починає з української, потім перекладає і нарешті говорить.

### 4) Учасник говорить англійською балакає безстрашно. Чи є помилки чи не має, те не стримує його від доведення своєї думки.

Учасник вільно говорит. Може бути невірна вимова, або обороти не. Можливі помилки у складно-підрядних реченнях та часах. Допомога від занять буде більше у тому щоб зберегти навички, та поліпшити знання.
